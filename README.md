# Fitness Coach

Status: Beta

## Deployment

Upload these files to the root of your GitHub repository:

- index.html
- manifest.json
- icon.png
- README.md
- FITNESS_COACH_CHANGELOG.md

## Overview

Fitness Coach is a personal fitness coaching application that includes:

- Personalized profile setup
- RPG-style fitness classes
- Avatar/body type selection
- Functional fitness assessment
- Goal weight tracking
- Workout coaching
- Pain and recovery tracking
- Supplement tracking
- Backup/export support

## Architecture

- Single-page application
- GitHub Pages compatible
- Minimal deployment structure
- Local browser storage
- No API keys or embedded secrets
